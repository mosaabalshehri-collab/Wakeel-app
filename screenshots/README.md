# لقطات الشاشة

ضع لقطات الشاشة هنا بهذه الأسماء بالضبط حتى تظهر تلقائياً في `README.md` الرئيسي:

| اسم الملف | الصفحة المطلوب تصويرها |
|---|---|
| `dashboard.png`  | لوحة التحكم `/dashboard` (الإحصائيات + تنبيه الانتهاء) |
| `poa-list.png`   | قائمة الوكالات `/poa` (مع شريط البحث والفلترة ظاهراً) |
| `poa-new.png`    | فورم إضافة وكالة `/poa/new` |
| `login.png`      | صفحة تسجيل الدخول `/login` |
| `dark-mode.png`  | أي صفحة في الوضع الداكن (لإظهار الـ Dark Mode) |
| `trash.png`      | سلة المحذوفات `/poa/trash` |

## نصائح للقطات احترافية

- استخدم عرض متصفح ~1280px، وكبّر الصفحة قليلاً (zoom 110%) لوضوح النص.
- املأ التطبيق ببيانات واقعية أولاً (3–5 وكالات بحالات مختلفة: سارية،
  منتهية، قاربت على الانتهاء) حتى تبدو اللوحة حيّة لا فارغة.
- استخدم صيغة PNG، وحاول إبقاء عرض كل صورة ≤ 1600px لتخفيف حجم الريبو.
- لإخفاء البيانات الحساسة، استخدم أسماء وأرقام وهمية.
